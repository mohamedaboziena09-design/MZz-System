Professional Desktop Management System developed by MZz Hub.

Created by Mohamed Ziena.

---

## Overview

MZz System is a desktop-based management platform designed to organize and manage people, employee records, vehicles, documents, and operational data through a unified interface.

The system is built with Python, SQLite, and CustomTkinter, following a structured versioning roadmap:

* V1.1.x Database Foundation
* V1.2.x API Foundation
* V1.3.x Search Engine
* V2 Security & Gate Operations
* V3 Intelligent Data Services

The current release includes a functional desktop application, executable packaging, employee management modules, vehicle management modules, reporting screens, and a centralized dashboard.
